### Manual Install

1. Install dependencies:

    - git
    - make
    - inkscape
    - xcursorgen

2. Run the following commands as normal user:

    ```
    git clone https://github.com/0jdxt/oreo-cursors.git
    cd oreo-cursors
    ./generate_colours.py
    make build
    sudo make install
    ```

3. Choose a theme in the Settings or in the Tweaks tool.
